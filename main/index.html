Freeflo is a tech startup with a mission of providing companies quality technical assistance one needs to reach out to a wider customer base and build their business.

B2C(Begin2Code)

B2C stands for Begin2Code.
B2C is a tool that we've built for coders to use in their beginning stages. One could use this tool to:
1) Indent their code properly
2) Execute their code without running command on command line
3) Perform a syntax lookup incase of any doubts
The code for this project can be found here.

Telegram SyntaxDB Bot

Telegram SyntaxDB is a bot that's been deployed on Telegram as the name suggests. This bot can be used by all Telegram users.
The main purpose of this bot is to perform and help you with syntax lookup on Telegram as nowadays, Telegram has become an app where most open programming groups are found.
This bot has been featured on Python India and Python Offtopic groups on Telegram. Click here to look at its code.

SyntaxDB Python API

SyntaxDB is a one stop solution for coders who have just started their journey in the world of coding. One can look for syntax related to various programming languages on this website.
We at FreeFlo, have created a python wrapper for this, in order to help developers integrate this feature of syntax lookup into their applications. The code for this project can be found here.

Thinkers Tree

Thinkers Tree is a website that we at FreeFlo have created for one of our clients(Ari Ari Business Solutions).
This website is basically an open research publication website that people can use to upload and go through research papers for free.
Yes, you read it right! FOR FREE.
Basically anyone who's interested to go through or submit their own research paper can sign up on this website. The research papers are published only after a thorough check by a panel comprising of experts in that domain.

FREEFLOEST. 2018

About the Freeflo Community

We at FreeFlo have started our own community called ChaoTech, an open source community that aims at highlighting the technological advancements made in today's world. It also looks at teaching people the basics of coding, machine learning, blockchain, data science and much more.

WHAT IS THE MOTTO OF CHAOTECH?
The primary motto of this community is not only to encourage everyone to get to know more about technology and understand how it makes our lives better but also to make sure that everyone's dream to learn about the latest technologies can actually come true.

WHY OPEN SOURCE COMMUNITY?
One of the main reasons why we've chosen to start this open source community is because we wanted to help people understand all the latest buzzwords like ML, AI, Blockchain etc., and making it an open source community actually gives us the freedom to explore all these fields. Moreover, we at FreeFlo are huge supporters of open source softwares. Infact, most of our projects at FreeFlo are open source projects that we constantly look to update in order to make the user experience better. We strongly believe that open source softwares are the way ahead as people are worried about their personal data being collected. Open source softwares help people to go through the code and understand whether their data is actually being collected or not. Also, from a developer's perspective, open source softwares not only help developers write better code but also helps developers with their code reading and debugging skills.

WHAT IS OUR VISION FOR CHAOTECH?
Our vision is to help our fellow and budding coders in different ways. In the coming days, as a part of this community, we'll be launching a tech blog where people can read about the latest technologies. Our team is going to set a good platform for developers as we'll constantly be uploading about problems faced by developers and how one could overcome them. We want to create an opportunity to get all tech folks under one roof so that everyone benefits. We look at making this community a one stop solution.

HOW TO JOIN CHAOTECH?
You can join our community by becoming a member of our meetup group on meetup.com. You can also open a college chapter for ChaoTech if your college doesn't have one already. All you need to do is fill the form here. We are here to clarify any queries regarding the community and you can reach us by writing to us at community@freeflo.tech
